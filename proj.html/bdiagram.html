What Behavioral Diagram?

USE CASE DIAGRAM

Use-case diagrams serve to model a system's behavior and to identify the system's requirements.

STATE MACHINE DIAGRAM

A state diagram in the Unified Modeling Language (UML) illustrates transitions between different objects and is also referred to as a state machine diagram.

INTERACTION DIAGRAM

The interactive behavior of a system is depicted in interaction diagrams, a particular kind of UML diagram.

COMMUNICATION DIAGRAM

To examine the dynamic behavior of a system or software application, you can use a particular kind of interaction diagram called a communication diagram.
